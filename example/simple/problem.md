# statement
$A + B$ を出力してください。

# input

```
$A$
$B$
```

$A,B$ は次の制約を満たします。

$0 \leq A < 10$  
$1000 \leq B \leq 10^{100}$


# output

$A + B$ を出力してください。 最後に改行してください。


# sample

```sample-input
1
1000
```

```sample-output
1001
```

1+1000 = 1001


```sample-input
0
10000000000000000000000000000000000000000000000000000
```

```sample-output
10000000000000000000000000000000000000000000000000000
```
